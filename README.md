# Akuma no Shima

This is the repository for the 2D side scroller game called Akuma no Shima (Demon Island).

Play as Noroi, the main character, as you are dropped into unknown territory, battling hordes of Oni and Hellhounds, increasing your abilities as you progress. There are several types of biomes that can generate, each with its own unique feature. Once all levels are cleared, the user can progress to the final showdown against Maou (Demon Lord). In the final battle, the user will be expected to overcome every challenge Maou throws at them with nothing but the skills they acquired and their tenacity.

Once the final battle has concluded, the user will unlock the endless mode to test their mettle, until they are ready to challenge the boss once more.

This game aims to utilize the following techniques:
- Procedural World Generation
- Unique character designs
